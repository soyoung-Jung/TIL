# 13. 메소드 + 접근자

> ### 메소드 Overloading

- 같은 이름의 메소드를 여러 개 선언하고 정의할 수 있다.

  조건: 매개 변수의 타입, 개수, 순서 중 하나가 달라야 한다.

> ### 메소드 접근자

- public

  다른 객체에서도 호출이 가능하다. 객체 생성 후 `객체.메소드()`형태를 띄어야 함. 

- private

  해당 메소드가 존재하는 객체 내에서만 호출이 가능하다.
